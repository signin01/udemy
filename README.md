## Udemy Webpage Clone

This is a basic Udemy webpage clone created using HTML and CSS. 
The project focuses on replicating the visual layout and structure 
of the Udemy homepage for learning purposes. It includes sections 
such as the header, navigation bar, course categories, featured 
courses, and footer.

### Features
- Single-page layout
- Navigation bar similar to Udemy homepage
- Course categories and featured courses sections
- Clean and responsive design using basic CSS
- Simple and well-structured HTML

### Technologies Used
- HTML5
- CSS3

### Purpose
This project was developed for educational purposes to practice 
front-end web development fundamentals and improve webpage layout 
and styling skills.
